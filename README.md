# EITFWH-DCU-GPU

**Language / 语言**: [中文](#中文) | [English](#english)

---

## 中文

**EITFWH-DCU-GPU** 是一款面向气动声学 / 流动噪声预测的 **FW-H 声学后处理软件**，基于 **Ffowcs Williams–Hawkings (FW-H)** 方程开发，用于模拟非定常流动产生的声波辐射与传播。

本版本在原有 MPI 多节点并行框架的基础上，进一步支持 **DCU/GPU 加速计算**，可显著提升大规模 FW-H 后处理任务的计算效率。

---

### 最新更新：DCU/GPU 加速

本版本重点更新了 **FW-H 积分阶段的 DCU/GPU 加速能力**。通过将计算密集型的声学积分过程卸载到 DCU/GPU 加速设备上，可有效提升长时间序列、大规模积分面和多观测点条件下的后处理效率。

该更新主要用于解决：

* FW-H 后处理计算耗时过长
* 积分面元数量大、时间序列长、观测点多
* 单节点 CPU 计算效率不足
* 单节点内存或单设备显存受限
* 大规模工程算例声学后处理效率低

---

### 主要特性

* 基于 **FW-H 声学类比** 的非定常声辐射预测
* 支持典型声源机制：

  * **单极子（Monopole）**
  * **偶极子（Dipole）**
  * **四极子（Quadrupole）**
* 支持二维 / 三维声学后处理
* 支持时域 / 频域 FW-H 计算
* **C++** 核心求解器
* 支持 **MPI 多节点并行**
* 支持 **DCU/GPU 加速计算**
* 支持大规模非定常数据的并行读取与分发
* 适用于大规模 CFD/LES/DNS 结果的声学后处理

---

### 适用场景

* 螺旋桨、旋翼、风扇、翼型等流动噪声预测
* 水下推进系统非定常载荷噪声计算
* 多观测点远场声压预测
* 长时间序列声学信号重构
* 大规模表面 / 体数据的 FW-H 并行后处理
* CPU 内存或 GPU/DCU 显存受限的工程级算例

## English

**EITFWH-DCU-GPU** is an **FW-H acoustic post-processing software package** for aeroacoustics and flow-induced noise prediction. It is developed based on the **Ffowcs Williams–Hawkings (FW-H)** equation to model sound radiation and propagation generated by unsteady flows.

This version extends the original MPI-based multi-node framework by introducing **DCU/GPU acceleration**, significantly improving the efficiency of large-scale FW-H acoustic post-processing.

---

### Latest Update: DCU/GPU Acceleration

This release introduces **DCU/GPU acceleration for the FW-H integration stage**. By offloading the computationally intensive acoustic integration process to DCU/GPU devices, the software can efficiently handle long time histories, large integration surfaces, and many observer locations.

This update is designed to address:

* Excessive runtime in FW-H post-processing
* Large integration surfaces, long time series, and many observers
* Limited efficiency of CPU-only computation
* Single-node memory or single-device GPU/DCU memory bottlenecks
* Inefficient acoustic post-processing for engineering-scale cases

---

### Key Features

* Unsteady acoustic radiation prediction based on the **FW-H acoustic analogy**
* Supports typical acoustic source mechanisms:

  * **Monopole**
  * **Dipole**
  * **Quadrupole**
* Supports 2D and 3D acoustic post-processing
* Supports time-domain and frequency-domain FW-H computations
* Core solver implemented in **C++**
* **MPI-based multi-node parallelization**
* **DCU/GPU-accelerated computation**
* Parallel reading and distribution of large-scale unsteady datasets
* Suitable for acoustic post-processing of large CFD/LES/DNS datasets

---

### Application Scenarios

* Flow-induced noise prediction for propellers, rotors, fans, and airfoils
* Unsteady loading noise from underwater propulsion systems
* Far-field acoustic prediction with many observers
* Reconstruction of long acoustic pressure time histories
* Parallel FW-H post-processing of large surface / volume datasets
* Engineering-scale cases limited by CPU memory or GPU/DCU memory
